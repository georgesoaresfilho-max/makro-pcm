Política de Privacidade — Makro PCM Bot

Última atualização: 26 de maio de 2025

1. Sobre este aplicativo

O Makro PCM Bot é um sistema interno da Makro Engenharia utilizado exclusivamente para registro de apontamentos de manutenção industrial via WhatsApp.

2. Dados coletados

Coletamos apenas:

  • Número de telefone do usuário
  • Conteúdo das mensagens enviadas ao bot (apontamentos de OS)
  • Data e hora dos apontamentos

3. Uso dos dados

Os dados são utilizados exclusivamente para atualização do sistema de controle de manutenção interno da Makro Engenharia. Não são compartilhados com terceiros.

4. Armazenamento

Os dados são armazenados em planilhas Google Sheets de acesso restrito aos colaboradores autorizados da Makro Engenharia.
